Detailed Description

@package structure

Secondary structures assignment
    1. 4-helices (H)
    2. isolated beta-bridge (B)
    3. beta-strands (E)
    4. 3-helices (G)
    5. 5-helices (I)
    6. n-turns (T)
    7.bends (S)
